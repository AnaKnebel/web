O que é uma regra CSS?

CSS é um comando do HTML que nos ajuda a definir a aparência de cada elemento HTML usado na página. E para que ele funcione, ele precisa seguir uma regra onde é composta po duas partes:

  • Seletor: Seleciona qual(is) elemento(s) que serão utilizados para aplicar a regra. Pode ser usado mais de um seletor na mesma regra, basta estar separando eles por uma vírgula;
  • Declaração: Onde será definido a aparência do elemento.

Para criar um CSS precisamos utilizar o seguinte elemento: <style>

Podemos utilizar ele no <head> (é mais comum) ou no <body> (utilizado para mudar a aparência de algumas partes do código). Exemplo utilizando no <head>:

 <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        <!-- início do CSS -->
        <style>
            ul{ <--- Seletor
            background-color: rgb(8, 224, 224); <--- Declaração
            border: 5px outset darkcyan; <--- Declaração
        }
        </style>
        <!-- fim do CSS -->
    </head>
    <body>
        <!-- Conteudo do HTML -->
        <ul>
            <li> Criação da Lista </li>
        </ul>
    </body>
    </html>

O exemplo mostrado acima, irá deixar a lista com um fundo colorido e com uma borda (Conforme está a tabela acima sobre a regra).
