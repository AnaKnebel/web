Como incluir CSS nos documentos HTML?

Existem 3 formas de incluir CSS nos documentos:

  1. CSS Inline;
  2. CSS Interno;
  3. CSS Externo.

CSS Inline

Este método aplica estilos diretamente a um elemento HTML usando o atributo <style>. É útil para estilos específicos de um único elemento, mas não é recomendado para projetos maiores. Exemplo:

 <p style="color: blue"> Esse texto ficará azul.</p>

CSS Interno

Use a tag <style> para inserir códigos CSS dentro da seção <head> do seu documento HTML. Este método é útil para estilos que se aplicam apenas a uma página específica. Exemplo:

    <head> 
        <style> 
            h1{
                color:green;
            }
        </style>
    </head>
    <body> 
        <h1>Este título é verde.</h1>
    </body>

CSS Externo

Este é o método masi comum e recomendado, especialmente para projetos maiores. Ele envolve a criação de um arquivo9 .css separado e a vinculação dele ao seu HTML usando a tag <link> na seção <head>. Exemplo:

No arquivo HTML:
            <head> 
                <link rel="stylesheet" href"style.css">
            </head>
No arquivo CSS (style.css):
            h1{
                color: red;
            }
