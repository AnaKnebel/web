O que é uma regra CSS?

Primeiramente para saber o que é uma regra CSS, vamos entender o que é CSS.

CSS (Cascanding Style Sheets) é um comando do HTML que ajuda a definir a aparência de cada elemento HTML da página.

Agora sim, o que é uma regra CSS? Para que o CSS funcione corretamente, ele precisa seguir uma regra que é composta por duas partes:

  • Seletor: seleciona quais são os elementos em que a regra será aplicada. Podendo usar vários seletores em uma mesma regra, separando eles com uma vírgula;
  • Declaração: define a aparência do elemento.

Definindo o Estilo (aparência)

Para criar um CSS precisamos utilizar o seguinte elemento: <style>

Podemos utilizar ele no <head> (é mais comum) ou no <body> (utilizado para mudar a aparência de algumas partes do código). Exemplo:

 <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        <!-- início do CSS -->
        <style>
            ul{
            background-color: rgb(8, 224, 224);
            border: 5px outset darkcyan;
        }
        </style>
        <!-- fim do CSS -->
    </head>
    <body>
        <!-- Conteudo do HTML -->
    </body>
    </html>

Seletores CSS

O seletor define a que(ais) elemento(s) HTML da página a regra CSS será aplicada. Há diversos tipos de seletores como por tag & id. Regras podem ter mais de 1 seletor, basta separar por vírgula.
