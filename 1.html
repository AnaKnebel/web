O que é HTML?

O HTML (HyperText Markup Language) foi criado por Tim Berners-Lee, um físico britânico, no CERN (Organização Europeia para a Pesquisa Nuclear), o HTML acabou surgindo através de uma necessidade urgente, para facilitar a troca de informaçãoes dentro de um ambiente acadêmico e para pesquisas colaborativas.

Um arquivo HTML é uma linguagem para a definição dos elementos gráficos de uma página web. Quando é feito uma solicitação de algum cliente, o servidor nos devolve uma resposta em HTML.

Um código HTML é criado através de tags, que são como etiquetas para informar qual o elemento que deve ser usado para o texto. Esse é o formato da Estrutura Padrão de um arquivo HTML:

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document
            </head>
            <body>
                
            </body>
            </html>

Algumas tags e suas funções:

  1. <head>: Essa tag informa o cabeçalho da página;
  2. <title>: O nome dado ao arquivo;
  3. <body>: Essa informa o corpo do documento;
  4. <p>: Tag para escrevermos um paragráfo;
  5. <br>: É usado para fazer uma quebra de linha;
  6. <h1> até <h6>: São modelos de Títulos para o documento;
  7. <hr>: Cria uma linha para separação na horizontal;

A linguagem do HTML que acabou construindo toda a WEB, permitindo a criação das páginas web que poderiam ser interligadas através de links.

O código HTML é essencial para que exiba o documento de um HTTP (é um protocolo de comunicação utilizado para a transferência de informações).
